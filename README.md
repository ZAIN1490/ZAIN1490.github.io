# Personal Portfolio Website

A sleek, premium, and fully responsive single-page personal portfolio website built to showcase my journey, technical skillset, and upcoming engineering projects. 

## 🚀 Live Demo
You can view the live deployment here: **[zain1490.github.io](https://zain1490.github.io/)** *(or insert your custom domain)*

## ✨ Key Features
* **Single-Page Smooth Scrolling:** Seamless navigation across distinct content sections using CSS logic.
* **Modern Floating Glassmorphic Navbar:** A frosted-glass sticky navigation hub featuring an intelligent color-mask shield to prevent text bleed-through during scrolling.
* **Typing Animation Hero:** A custom CSS-driven micro-animation engine rendering a typewriter style title introduction.
* **Horizontal Tech Stack Grid:** An elevated grid layout showcasing active skill proficiencies with vertically stacked brand icons and typography.
* **Responsive Architecture:** Handcrafted responsive styling built for screens ranging from mobile phones to high-resolution desktop frames.

## 🛠️ Tech Stack & Tools
* **Core:** HTML5, CSS3 (Custom Flexbox Grid & Animations)
* **Typography:** Inter Font Family (via Google Fonts)
* **Iconography:** Font Awesome v6.5.1
* **Version Control:** Git & GitHub
